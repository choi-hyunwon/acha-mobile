    • 해외 이용시 청구금액 산출방법

    • * 해외 이용시 청구금액 = (거래미화금액 x 전신환 매도율¹) + 국제 브랜드 수수료² + 해외서비스 수수료³
    • 1. 전신환 매도율 : 접수일의 KB국민은행 최초고시 전신환 매도율
    • 2. 국제브랜드수수료 = (거래미화금액 x 국제브랜드 이용수수료율) x 전신환 매도율
    • 3. 해외서비스수수료 = (거래미화금액 x 해외서비스 수수료율) x 전신환 매도율
